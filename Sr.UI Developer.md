# Senior UI Developer Interview Questions & Answers

## Overview

This repository contains comprehensive interview preparation materials for Senior UI Developer positions. The content has been organized into modular, topic-specific guides for easy navigation and reference.

## Table of Contents

### Core Web Technologies
- [HTML & CSS](./HTML-CSS-Interview-Guide.md) - 15 questions covering semantic HTML, responsive design, box model, specificity, Flexbox, Grid, accessibility, and more
- [SCSS (Sass)](./SCSS-Interview-Guide.md) - 10 questions on variables, nesting, mixins, maps, BEM methodology, and performance optimization
- [JavaScript](./JavaScript-Interview-Guide.md) - 8 questions on closures, event delegation, promises, async/await, event loop, and prototypal inheritance
- [TypeScript](./TypeScript-Interview-Guide.md) - 20 comprehensive questions covering static typing, interfaces vs types, generics, utility types, union types, mapped types, discriminated unions, Vue component props, composables, Pinia stores, Nuxt API endpoints, type-safe routing, TypeScript configuration, declaration files, template literal types, event emitters, form handling, and performance optimization

### Frontend Frameworks
- [Vue.js](./Vue-Interview-Guide.md) - 11 questions covering Composition API, lifecycle hooks, state management with Pinia, API calls, authentication, error handling, pagination, and infinite scrolling
- [Nuxt.js](./Nuxt-Interview-Guide.md) - 13 questions on SSR/SSG, page-based routing, middleware, composables, modules, API routes, authentication, WebSockets, caching strategies, and performance optimization
- [React.js](./React-Interview-Guide.md) - Comprehensive guide covering component lifecycle, hooks, controlled/uncontrolled components, Virtual DOM, Context API, performance optimization, HOCs, form handling, and error boundaries
- [Next.js](./Next-Interview-Guide.md) - Complete guide on data fetching strategies, ISR, image optimization, authentication patterns, middleware, Pages Router vs App Router, and SEO implementation

### Modern Development Practices
- [AI-Assisted Development](./AI-Assisted-Development-Guide.md) - 8 questions on using AI coding assistants, prompt engineering, code validation, test generation, limitations, debugging assistance, refactoring, and CI/CD integration
- [AI-Assisted Debugging Scenarios](./AI-Debugging-Scenarios.md) - 4 live coding scenarios with problematic code and solutions:
  - Scenario 1A: Vue 3 memory leak and reactivity bug
  - Scenario 1B: Nuxt 3 SSR issues and composable misuse
  - Scenario 2: TypeScript unsafe types and async pitfalls
  - Scenario 3: HTML/CSS accessibility and layout bugs
  - Scenario 4: Performance optimization for large datasets
